# Persistent Sessions with tmux

Your terminal runs inside tmux. Sessions survive disconnects.

## Basic Commands

- `Ctrl+b d` - Detach
- `Ctrl+b c` - New window
- `Ctrl+b n` - Next window
- `Ctrl+b p` - Previous window
- `Ctrl+b [` - Scroll mode

## Workflow

1. Start long-running task (e.g., `npm run dev`)
2. Close browser
3. Return hours later
4. Session resumes automatically
